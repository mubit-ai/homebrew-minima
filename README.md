# Mubit-ai Minima

## How do I install these formulae?

`brew install mubit-ai/minima/<formula>`

Or `brew tap mubit-ai/minima` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "mubit-ai/minima"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
